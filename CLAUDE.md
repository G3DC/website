# CLAUDE.md — G3 Website

## Project overview

Coming-soon landing page for **Global Green Grid (G3)**, a data centre infrastructure company. Built with React + Vite + TanStack Router.

## Commands

```bash
bun dev          # Start dev server
bun build        # Type-check + build
bun lint         # ESLint
bun format       # Prettier (also sorts Tailwind classes)
bun typecheck    # TypeScript only
bun preview      # Preview production build
```

## Stack

| Concern | Library |
|---|---|
| Framework | React 19 |
| Routing | TanStack Router (file-based, auto code-splitting) |
| Styling | Tailwind CSS v4 + tw-animate-css |
| UI primitives | shadcn/ui (Radix UI + Base UI) |
| Animation | Framer Motion |
| Icons | Lucide React |
| Theme | Custom `ThemeProvider` (light / dark / system, persisted to `localStorage`) |

## Folder structure

```

│   ├── custom-ui/
│   │   ├── animation/
│   │   │   └── Synapse.tsx    # Hero SVG neural-network animation (Framer Motion)
│   │   └── home/
│   │       ├── NavBar.tsx
│   │       ├── HeroSection.tsx
│   │       └── Footer.tsx

├── pages/

```

## Routing

- File-based routes live in `src/routes/`. `routeTree.gen.ts` is auto-generated — never edit.
- `__root.tsx` is the root layout; it wraps `<Outlet />` in `<ThemeProvider>`.
- Route components are registered directly (e.g. `component: HomePage`) — no extra wrapper divs.
- `App.tsx` is kept as a Vite project convention but does nothing.

## Path alias

`@/` → `src/`. Always use it — no relative `../../` imports.

## Styling conventions

- **Tailwind v4** — all config lives in `src/index.css` via `@theme`. There is no `tailwind.config.js`.
- **Colors** use OKLCH. Semantic tokens (`--color-background`, `--color-foreground`, etc.) swap between light/dark via `@custom-variant dark` (class-based: `.dark` on `<html>`).
- Use `cn()` from `@/lib/utils` to merge Tailwind classes.
- Use `cva` for component variants.
- Run `bun format` before committing — Prettier enforces Tailwind class order.
- No semicolons, double quotes, 80-char line width (see `.prettierrc`).

## Theme

- `ThemeProvider` defaults to `"system"`, reads/writes `localStorage` key `"theme"`.
- `useTheme()` is available anywhere inside the provider tree.
- The **Footer** exposes a `Sun / Monitor / Moon` icon toggle group (no background boxes — icons only, active item is `teal-400`).
- On mobile the theme toggle is commented out from the footer's top row to save space — only shown on desktop.

## Component notes

### NavBar
- Floating pill, fixed positioned, `z-50`.
- "Global Green Grid" label: `whitespace-nowrap` single line at `10px` on mobile; three-line vertical block at `7px` on desktop.
- Pipe between G3 and label uses `bg-foreground/20` (visible in both light and dark).

### HeroSection
- Layout: `flex flex-col min-h-svh` on the `HomePage` wrapper — `flex-1` on `<main>` fills remaining space.
- Top padding (`pt-24 md:pt-28`) clears the fixed navbar.
- Synapse animation: `absolute`, `md:top-[92px]` (below navbar bottom ~88px), `md:right-[2.5%]` (aligned with navbar right edge), `md:w-[38%]`.
- "AI . DATA CENTRE . INFRASTRUCTURE" tagline: `text-[14px]`, `teal-900` light / `teal-700` dark.
- No up/down float on the animation — removed intentionally.

### Synapse (animation)
- Pure SVG + Framer Motion, no external data.
- Outer ring pulses outward (`scale 1→1.06`, `opacity 0.6→0.15`) as a wave/ripple.
- Blinking LED dot in the server rack centre is `#2dd4bf` (teal-400).

### Footer
- Always `bg-teal-950` regardless of theme.
- **Mobile** (two rows):
  - Row 1: branding (G3 + logo) left — theme toggle right *(toggle currently commented out)*
  - Row 2: Privacy/Terms/Contact left — abbreviated copyright right
- **Desktop** (single row): branding → links → theme toggle → copyright (`justify-between`).
